# i18n

## Text domains

- **Free plugin:** `wedevs-project-manager` (must match `Text Domain:` in `cpm.php` plugin header).
- **Pro plugin:** `pm-pro` (separate; managed in `../pm-pro/`).

Always pass the literal text domain string — never a variable, never concatenated. WP scanners parse the literal at build time.

## React / JS

- Use `@wordpress/i18n`:
  ```jsx
  import { __, _n, _x, sprintf } from '@wordpress/i18n'
  __('Save', 'wedevs-project-manager')
  sprintf(__('Showing %1$d of %2$d tasks', 'wedevs-project-manager'), shown, total)
  _n('1 task', '%d tasks', count, 'wedevs-project-manager')
  ```
- Locale data is fed at boot in `views/assets/src/index.jsx`:
  ```js
  import { setLocaleData } from '@wordpress/i18n'
  setLocaleData(PM_Vars.language.pm.locale_data['wedevs-project-manager'], 'wedevs-project-manager')
  ```
  PHP injects `PM_Vars.language.pm.locale_data` (the JED format).

- Do NOT use a custom `useI18n` wrapper — recent commit `30d81498` removed it. Always import directly from `@wordpress/i18n`.

## PHP

```php
__('Save', 'wedevs-project-manager');
_e('Save', 'wedevs-project-manager');
_n('1 task', '%d tasks', $count, 'wedevs-project-manager');
sprintf(__('Welcome, %s', 'wedevs-project-manager'), esc_html($name));
```

## Generating POT

```bash
pnpm makepot
# => languages/wedevs-project-manager.pot
```

Excludes: `node_modules, build, tests, views/assets/dist, vendor`.

## Catalogs

- `.po` / `.mo` per locale in `languages/`.
- Fuzzy entries must be stripped before commit (recent commit `806fad1a`).
- `.json` files (per script handle) feed `wp_set_script_translations` — confirm the handle matches what PHP enqueues.

## Build-time JSON map

`build-i18n-map.mjs` validates the sourcemap shape (commit `bbe64962`). If you touch the bundle entry or shape, re-run and confirm the validator passes.

## Anti-patterns

- ❌ Variable text domain: `__('x', $domain)`.
- ❌ Concatenated text domain: `__('x', 'wedevs-' . 'project-manager')`.
- ❌ Interpolating user data into the translatable string: use `sprintf`.
- ❌ Translating fragment tokens (`__('Save ')` + `__('Task')`) — translate the full sentence.
