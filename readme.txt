== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from the Plugins page.

== Frequently Asked Questions ==

= Q. Why doesnt it show up in frontend =
A. Currently all the project management options reside in the admin panel. No functionality shows up in the front end right now.

= Q. Who can create projects? =
A. Only Editors and admins can create projects and edit them.

= Q. Which Project Management Tool is best for your need? =
A.  We have prepared a comparison of popular WordPress project management plugins. Here is the comparison table. We hope that will help you choose the best one for your needs. 

= Q. Does WP Project Manager plugin solve all project management challenges? =
A.  None of the tool can solve every problems but we can assure that our WP Project Manager Plugin will solve possible popular project managent chalenges. Get the detailed overview from here. 

= Q. How to manage tasks with WP Project Manager =
A. We have prepared a detailed tutorial for your. Watch it now - 
[youtube https://www.youtube.com/watch?v=ifmgV2kKfEk&t=136s&ab_channel=weDevs]

= Q. Who can create a task list, to-do, messages, or milestones? =
A. Admins and every co-worker from a project can create these.

= Q. Is there way to get the overview of WP Project Managers Gantt Chart feature =
A. Of course, please check our this overview video of Gantt Chart feature - 
[youtube https://www.youtube.com/watch?v=KUGywSKQsF0&ab_channel=weDevs]

= Q. How to track the progress of a task? =
A. You can easily track the progress of the task of your team member from task overview report. Here is the overview of that - 
[youtube https://www.youtube.com/watch?v=ec-4TXP7o7w&ab_channel=weDevs]

= Q. Can any use WP Project Manager as Trello or Trello like Kanban Board =
A. WP Project manager has Kanban board module. You can easly manage your tasks with that view. Here is the video overview - 
[youtube https://www.youtube.com/watch?v=5bhmpalVs7E&ab_channel=weDevs]

= Q. Can every member see every project? =
A. Only admins (editor/administrator) can see all of them. Other user roles can only see their assigned projects.

= Q. Can the WP Project Manager plugin be extended? =
A. Sure, lots of actions and filters are added by default, and will add more of them.
